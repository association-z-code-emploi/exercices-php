# Utiliser des tableaux

## Contexte et Objectifs

Utilisons maintenant des variables plus complexes... des tableaux !

## Spécifications

Créons différentes variables :

- un **tableau** contenant quatre **nombres entiers** ;
- un second **tableau** contenant quatre **nombres décimaux** ;
- un troisième **tableau** contenant deux **strings** : votre prenom et votre nom de famille

Affichez ensuite dans le terminal :

- la **somme** des premières valeurs du premier et second tableaux ;
- la **soustraction** des deuxièmes valeurs du premier et second tableaux ;
- le **produit** des troisièmes valeurs du premier et second tableaux ;
- la **division** des quatrièmes valeurs du premier et second tableaux ;
- la **concaténation** des deux valeurs du troisième tableau

## Conseil

- Utilisez `echo` pour afficher le contenu de vos variables.
- Codez directement dans le fichier "index.php" de ce dossier "01_variables/04_utiliser_des_tableaux".
- Pour exécuter votre fichier "index.php" :
  - rendez dans votre terminal ;
  - déplacez-vous dans le dossier où se trouve votre fichier ;
  - puis tapez `php index.php`
