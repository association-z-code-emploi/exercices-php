# Les 4 saisons

## Contexte et objectifs

Codez une condition vous permettant de connaître la saison selon le mois.

Pour rappel (et pour vous simplifier la tâche) :

- mars, avril et mai → printemps
- juin, juillet et août → été
- septembre, octobre et novembre → automne
- décembre, janvier et février → hiver

## Spécifications

Créez une variable `currentMonth` contenant une string représentant le mois actuel.

Codez une condition qui affichera la saison correspondante à la valeur de `currentMonth`.

Par exemple, si la valeur de `currentMonth` est "octobre", alors on affichera :

```
Nous sommes en automne.
```

## Conseil

Il existe des fonctionnalités intégrées à PHP qui permet de donner la date actuelle, dont le mois. Néanmoins, pour cet exercice, écrivez simplement le mois avec une string. Cela vous permettra aussi de changer cette valeur pour tester les autres sorties de votre condition !
